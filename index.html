注意事项

隐私与权限

您的财务隐私仅属于您自己。应用不需要任何敏感权限（如定位、联系人等）。数据通过 Room 数据库存储在手机私有沙盒中。我们不设服务器，绝不上传记录，完全离线运行。

【必读‼️】如何防止数据丢失？

✅ 覆盖安装：从 GitHub 下载新版本后直接点击“更新”。系统会替换代码并保留您的账单数据库。

❌ 严禁卸载：请勿先卸载旧版本。Android 卸载时会同步抹除所有私有数据，且无法找回。
